Lightweight PHP MVC Framework with PDO and Closures

ToePrint is a minimalist approach at building a powerful, flexible, development-centered PHP5 Framework leveraging PDO and closures, or lambda functions. ToePrint is still very young, and some of the features are still being completed, but it is functional without modification as a lightweight PDO-MySQL powered MVC skeleton. ToePrint Framework does not currently support automatic file includes, but examples will be provided to enable this functionality.
